Python Kodlama Örnekleri

Python ile basit bir "Merhaba Dünya" örneği:

print("Merhaba, Dünya!")

Python, okumayı ve yazmayı kolaylaştıran, güçlü ve popüler bir programlama dilidir. İşte birkaç Python örneği:

Python - Toplama İşlemi

a = 5
b = 10
print(a + b)

Python - Döngü Kullanımı

for i in range(5):
    print(i)

HTML Kodlama Örnekleri

HTML ile basit bir web sayfası örneği:

<!DOCTYPE html>
<html>
  <head>
    <title>Web Sayfası</title>
  </head>
  <body>
    <h1>Merhaba, HTML!</h1>
  </body>
</html>

HTML, web sayfalarını oluşturmak için kullanılan temel bir işaretleme dilidir. İşte birkaç HTML örneği:

HTML - Form Örneği

<form>
    <label>Adınız:</label>
    <input type="text">
    <button type="submit">Gönder</button>
</form>

C# Kodlama Örnekleri

C# ile basit bir konsol uygulaması örneği:

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Merhaba, C#!");
    }
}

C# güçlü ve modern bir dil olup, .NET framework üzerinde çalışır. İşte birkaç C# örneği:

C# - Basit Hesap Makinesi

using System;

class Program
{
    static void Main()
    {
        int a = 10, b = 5;
        Console.WriteLine(a + b);
    }
}

C++ Kodlama Örnekleri

C++ ile basit bir "Merhaba Dünya" örneği:

#include <iostream>
using namespace std;

int main() {
    cout << "Merhaba, C++!" << endl;
    return 0;
}

C++, çok güçlü ve performans odaklı bir programlama dilidir. İşte birkaç C++ örneği:

C++ - Dizi Kullanımı

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}
