Python Kodlama Örnekleri

Python, öğrenmesi kolay, güçlü ve çok yaygın bir programlama dilidir. İşte bazı temel Python örnekleri:

Python - Merhaba Dünya

print("Merhaba, Dünya!")

Python - Toplama İşlemi

a = 5
b = 10
print(a + b)

Python - Döngü Kullanımı

for i in range(5):
    print(i)

HTML Kodlama Örnekleri

HTML, web sayfalarının temel yapı taşlarını oluşturan işaretleme dilidir. İşte bazı temel HTML örnekleri:

HTML - Merhaba Dünya

<!DOCTYPE html>
<html>
  <head>
    <title>Web Sayfası</title>
  </head>
  <body>
    <h1>Merhaba, HTML!</h1>
  </body>
</html>

HTML - Form Örneği

<form>
    <label>Adınız:</label>
    <input type="text">
    <button type="submit">Gönder</button>
</form>

C# Kodlama Örnekleri

C#, .NET Framework ile çalışan güçlü bir dildir. İşte bazı temel C# örnekleri:

C# - Merhaba Dünya

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Merhaba, C#!");
    }
}

C# - Basit Hesap Makinesi

using System;

class Program
{
    static void Main()
    {
        int a = 10, b = 5;
        Console.WriteLine(a + b);
    }
}

C++ Kodlama Örnekleri

C++, çok güçlü ve esnek bir dil olup, sistem programlamada yaygın olarak kullanılır. İşte bazı C++ örnekleri:

C++ - Merhaba Dünya

#include <iostream>
using namespace std;

int main() {
    cout << "Merhaba, C++!" << endl;
    return 0;
}

C++ - Dizi Kullanımı

#include <iostream>
using namespace std;

int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}
