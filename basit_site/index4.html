Kodlama Öğrenmeye Başla!

Python, HTML, C#, C++ ve daha fazlasını öğrenmek için örneklerle dolu platform.

Python Kodlama Örnekleri

Python, okunabilirliği kolay ve güçlü bir dildir. İşte başlangıç için bir örnek:

print("Merhaba, Dünya!")

Bu basit kod, ekrana "Merhaba, Dünya!" yazdırır.

HTML Kodlama Örnekleri

HTML, web sayfalarını oluşturmak için kullanılan temel bir dildir. İşte basit bir HTML örneği:

<!DOCTYPE html>
<html>
  <head>
    <title>Web Sayfası</title>
  </head>
  <body>
    <h1>Merhaba, HTML!</h1>
  </body>
</html>

Bu örnek, basit bir web sayfası oluşturur ve başlık olarak "Merhaba, HTML!" yazdırır.

C# Kodlama Örnekleri

C# güçlü ve modern bir dildir. İşte basit bir C# kodu:

using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Merhaba, C#!");
    }
}

Bu örnek, "Merhaba, C#!" mesajını ekrana yazdırır.

C++ Kodlama Örnekleri

C++, çok yönlü ve güçlü bir dil olarak geniş bir kullanım alanına sahiptir. İşte basit bir C++ örneği:

#include <iostream>
using namespace std;

int main() {
    cout << "Merhaba, C++!" << endl;
    return 0;
}

Bu örnek, "Merhaba, C++!" mesajını ekrana yazdırır.
